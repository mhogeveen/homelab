# Homelab

Anything and everything regarding my homelab setup.

## Technologies

- **Proxmox** - Open source server virtualization management
- **Tailscale** - Mesh VPN
- **Ansible** - Configuration management
- **Just** - Task runner

## Services

- [**Home Assistant**](https://www.home-assistant.io/) - Home automation
- [**Glance**](https://github.com/glanceapp/glance) - Customizable dashboard

## Prerequisites

- Ansible
- Just
- SSH access to hosts

## Usage

```bash
just configure-system   # General host configuration
just sync-files         # Synchronize host configuration files
```
